# Project Brief

## Project Name
AndroidTVMCP - Android TV Remote Control to MCP Bridge

## Core Purpose
This project is a bridge that exposes Android TV remote control functionality with MCP (Model Context Protocol) servers.The project is implemented in python and is a wrapper to functionlity of androidtvremote2 python library.
